Introduction

Admit it, you really want to go to MineCon, but doesn't it suck that MineCon is invite only? What if there was something just like MineCon, but how do I put this, less exclusive? Something that anyone and everyone could go to, no matter your age, location, or language you speak. CrafterCon is the first unofficial Minecraft Convention were all this is possible. It's all free and all online.

How does it work?

It's easy!
There are different things you can do to support or be part of the event:
-- Host a Public Server
1) You make a Minecraft Server (How to)
2) You submit the Server (Here)

-- Host a Private Server

2) You submit the Server (Here) (Don't worry, this is only for records, it won't be public for others to see)
3) You call your friends over, or meet up with them somewhere and play on a server
